This is a project created for Qustodio kids app.
The project consists of 2 example test cases (Coded) and some test cases written in gherkin (Yet to be coded)

The test scripts are based on behave framework and uses appium for creating connections with emulator.

The script can be run by calling  command - "behave --no-capture -f plain"

Also script can be called by calling runner.py by giving parameter --testdir=*path_to_dir*
